Fanzine augmenté

DNA 2020

Mon parti-pris était de construire un fanzine metal « augmenté » par le webtoprint qui permet deux niveaux de regard. J’ai fait en sorte qu’il soit possible de reprendre contrôle sur le contenu, permettant une mise en page différente à chaque impression. Ainsi le positionnement des éléments n’est pas totalement déterminé par le code puisqu’il est possible d’agir directement dessus, de revenir sur sa mise en page, ce qui évoque les techniques du fanzine DIY construit à la photocopieuse.
Travail initié lors de workshops avec Lucille Haute et Julie Blanc (janvier 2020) et avec Quentin Juhel (avril 2020)

Project executed as part of my Bachelor’s Degree in Visual and Graphic Design, 2020
This « augmented » Web to print metal fanzine allows two understanding layers, as a web page containing audio and video content, and as a printed object. The user is encouraged to regain control of the fanzine content, picking the position of the images. Each print has the potential to be unique as one can directly interact with the elements and impact the layout at any given moment before hitting « Print », which reminds of DIY copy-machine printed fanzines.
Project initiated during workshops with Lucille Haute and Julie Blanc (January 2020) and Quentin Juhel (April 2020)
